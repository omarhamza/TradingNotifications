# Local Development Setup

## Prerequisites

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/) (recommended version: 11)

## Run Locally

1. **Install Azurite**  
   Azurite is a local emulator for Azure Blob, Queue, and Table Storage. Install it globally using npm:

   ```powershell
   npm install -g azurite
   ```

2. **Run Azurite**

   Azurite is a local emulator for Azure Blob, Queue, and Table Storage. Install it globally using npm:

   ```powershell
   azurite
   ```
